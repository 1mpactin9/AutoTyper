## Fully Automated
The Script: [Tampermonkey Script.js](/scripts/advanced.js)
Beta Updates: [Latest Script.js](/scripts/beta.js)

### Features

- Lesson scraping
- Automatically skip unexecutable lesson
- 97% WPM target configuration
- 99.7% Accuracy configuration
- Clean user interface

### How to use?

1. Install [Tampermonkey](https://www.tampermonkey.net/) extension
2. Enable required premissions
3. Create, Save, and Enable
4. The UI should show up
5. Modify settings scrap
6. start!

> [!NOTE]
> Please go to extensions > manage extensions and then enable "Allow user scripts" then refresh the page to run the script successfully

## Single Lesson
The Script: [Console Script.js](/scripts/basic.js)

### Features

- Simple and quick to use
- Uses the newest key event simulation
- Configurable speed and accuracy
- Mimic human typing behavior

### How to use?

1. Open a lesson in Typing Club
2. Open DevTools > Console
3. Paste in the script
4. Enter!

Within the script, changing the constants as shown in the script changes the speed and accuracy. The preset is aiming for around 90 WPM for [Typing Club](typingclub.com).

> [!IMPORTANT]
> Users MUST turn off block on error(s) feature in Typing Club to ensure consistency of fake accuracy implementation
